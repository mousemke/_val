DOGE MODULE

These command all interact with a dogecoin wallet. Users must be identified with https://github.com/mousemke/nickserv or a compatible NickServ bot in order to use these commands, unless autoAuth is emable in the config file. (use autoAuth with care, as it skips all user verification)


doge (number)

Displays the current price of 1 Doge in Satoshi. If a number is given it displays the price of that many.


market

Displays the current price of 1 Doge in GBP, USD, XRP, CNY, EUR, AUD, LTC and Satoshi. If a number is given it displays the price of that many.


tip (user) (number)

Removes the given number of Doge from the sending user and gives them to the written user if the sending user is identified.


withdraw (address) (amount)

Withdraws the given amount to the given address


balance

Displays the number of Doge in the sending users account


deposit

Displays a deposit address specifically for the sending user to go into their account


active

Displays the number of active users


soak (number)

Removes the given number of Doge from the sending user and splits it evenly between all active users if the sending user is identified.




WORDS MODULE

Works with the Wordnik API to define words and run the unscramble game. Translations come from the MyMemory API


(2 letter code) (word or phrase) (2 letter code)|(2 letter code) (word or phrase)

Translates the given text to the given language from english. If 2 codes are given seperated by a pipe, the first is the source language, the second is the destination language.
2 letter language codes: http://www.kwintessential.co.uk/translation/articles/iso-language-codes.html


def (word or phrase) define (word or phrase)

Displays the definition of the supplied text


unscramble

Displays the current unscramble scores


word whirred

Displays the current unscramble word
* this command is only available in the channel defined by userConfig.unscramble


newWord

Votes to get a new unscramble word
* this command is only available in the channel defined by userConfig.unscramble




ANAGRAMM MODULE

Anagramm is a mixup of the words module, using the translations to translate the word into German before scrambling it. The definitions are returned in english, as well as a copy of the original english word. This could theoretically be expanded to any language desired.


anagramm

Displays the Anagramm scores


wort

Displays the current Anagramm word
* this command is only available in the channel defined by userConfig.anagramm


neuesWort

Votes to get a new Anagramm word
* this command is only available in the channel defined by userConfig.anagramm




BEATS MODULE

Contains all functions relating to Beat time


beat time

Displays the current beat (https://en.wikipedia.org/wiki/Swatch_Internet_Time)




FOURSQUARE MODULE

Using the Foursquare API, this grabs info about the area surrounding the latitude and longitude supplied in the config.


lunch (query) feedme (query)

Gets a random eatery from the area around the latitude and longitude supplied in the config. If there is a query written it will search for that in addition to location.
